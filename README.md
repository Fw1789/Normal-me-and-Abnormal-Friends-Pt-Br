# Normal Me and Abnormal Friends (PT-BR)

Este repositório contém os arquivos de localização para a visual novel **Normal Me and Abnormal Friends** mobile. O objetivo deste projeto é fornecer uma tradução de alta qualidade, focada em diálogos naturais e adaptação cultural para brasileiros.(A tradução foi feita, utilizando-se de Ia.)

- **Interface:** Traduzida

### 📥 Como aplicar a tradução (Passo a Passo)

Este patch consiste em arquivos `.csv`. Para aplicar no seu jogo, siga estas instruções usando o **MT Manager**:

1. **Baixe o APK original** do jogo "Normal Me and Abnormal Friends(1.0.3)".

2. **Localize o arquivo:** Baixe o arquivo `.csv` correspondente ao capítulo que deseja (ex: `chapter1.csv`) na aba [Releases].

3. **Abra o APK:** No MT Manager, clique no APK do jogo e selecione **"View"** (Visualizar).

4. **Caminho dos arquivos:** Navegue até a pasta onde ficam os scripts de texto (no caso é em `/assets/www/timeline/).

5. **Substituir:** Encontre o arquivo original com o mesmo nome e substitua-o pelo arquivo traduzido que você baixou aqui.

6. **Finalizar:** Saia da visualização, o MT Manager perguntará se deseja atualizar o APK. Diga que **Sim** e certifique-se de que a opção **"Auto-sign"** (Assinar automaticamente) esteja marcada.

7. **Instalar:** Instale o APK modificado e divirta-se!
