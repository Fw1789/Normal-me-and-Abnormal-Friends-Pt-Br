# Normal Me and Abnormal Friends (PT-BR)

Este repositório contém os arquivos de localização para a visual novel **Normal Me and Abnormal Friends**. O objetivo deste projeto é fornecer uma tradução de alta qualidade, focada em diálogos naturais e adaptação cultural para brasileiros.(A tradução foi feita, utilizando-se de Ia.)

## 📊 Status do Projeto

- **Capítulo 2:** 10% (Tradução iniciada)
- **Interface:** Traduzida
